# 🔍 AI Fraud Detection System

Upload financial transaction data (CSV/Excel) and AI flags suspicious transactions with a risk score, reasoning, and recommended action — High/Medium/Low risk classification.

## Tech Stack
Python, Streamlit, Groq (Llama 3.3), Plotly, Pandas

## How to Run
1. `pip install streamlit groq python-dotenv pandas plotly`
2. Add your Groq API key to `.env`: `GROQ_API_KEY=your_key`
3. `streamlit run app.py`
